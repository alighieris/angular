Angular CLI

Ferramenta de linha de comando que permite trabalhar com o Angular através de comandos que facilitam o desenvolvimento e execução de ações e tarefas rotineiras.

O CLI não faz parte do framework Angular, é uma ferramenta complementar do ecossistema Angular de desenvolvimento.

Para instalar o Angular CLI globalmente na máquina, executar: 

npm install -g @angular/cli

Para invocar a ferramenta de CLI, utiliza-se o executável ng.
Para ajuda: ng --help

Estrutura do projeto Angular
...

Pastas:
...

  • .angular/ ->
  • node_modules/ -> Pasta com módulos e dependências necessárias para o desenvolvimento da aplicação.
  • src/ -> Pasta do código fonte
    • Pastas:
      • environments/ -> Pasta com as configurações de variáveis de ambiente (de produção e de desenvolvimento)
      • assets/ -> pasta para mídias
      • app/ -> Contém os componentes do Angular
    • Arquivos:
      • polyfills.ts -> arquivo com configurações para browsers mais antigos e compatibilidade.
      • test.ts -> Arquivo de configuração dos testes
      • styles.css -> arquivos com estilos globais.
      • main.ts -> arquivo de entrada
      • index.hmtl -> arquivo base do html

Arquivos no root:
...

  • karm.conf.js -> Arquivo de configuração do Karma, dependência executora de testes.
  • angular.json -> Contém configurações do projeto Angular.
  • .browserslistrc -> Contém a lista de browsers que suportam o Angular

Criar novos componentes
...

Para criar um novo componente, automaticamente pela CLI usa-se o comando: 

    ng generate component <nome_componente>

    # ou a forma curta:

    ng g c <nome_componente>
